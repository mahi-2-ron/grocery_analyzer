# PureScan | Food Ingredient Analyzer

A premium, high-performance web application designed to analyze food ingredients for potential health risks. Built with React, TypeScript, and Framer Motion, it provides a stunning visual interface for understanding exactly what's in your food.

## ✨ Features

- **Ingredient Extraction**: Automatically parses complex label text into a clean list of ingredients.
- **Risk Assessment**: Identifies controversial additives, high sugars, and harmful preservatives.
- **Health Scoring**: Generates a dynamic health score (0-100) based on ingredient quality.
- **Smart Verdicts**: Provides actionable recommendations and clear warnings for each product.
- **Rich Aesthetics**: Dark mode, glassmorphism UI, and smooth micro-animations.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone or download the repository.
2. Run `npm install` to install dependencies.

### Development
Run the dev server:
```bash
npm run dev
```

### Build
Generate a production-ready bundle:
```bash
npm run build
```

## 🛠️ Technology Stack
- **Frontend**: React + TypeScript
- **Styling**: Vanilla CSS with Design Tokens
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
